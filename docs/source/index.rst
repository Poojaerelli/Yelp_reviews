Welcome to Yelp_reviews's documentation!
========================================

This documentation is about yelp dataset. Yelp dataset consists of reviews for different businesses sorted by many attributes. You can download it from `here`_. After downloading, you have to extract it twice to get the original dataset consisting of five json files.

Here, you will be going through the following steps:
 1. performing sentiment analysis on yelp reviews dataset.
 2. plotting the reviews according to city and business using plotly.

.. _here: https://www.yelp.com/dataset
